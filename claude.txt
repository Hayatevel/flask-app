Python のフレームワーク "Flask" を使用して動的Webアプリケーションを作成したいです
以下の要件の通りに作成の手伝いをして欲しいです。

開発環境:
- OS: macOS/Windows11
- Python: version 3.12.3
- Flask: version 2.3.3
- IDE: VS Code
- Project Name: flask-app

アプリ機能:
- ユーザーのアカウント作成機能とログインログアウトの機能
- テキスト感情分析アプリをメインの機能をする
- オープンソースの自然言語処理ライブラリをAPIを利用して使用
- ユーザーが入力したテキストの感情（ポジティブ/ネガティブなど）を分析し動的に出力

要件:
- お金の掛からない無料のリソースを使うこと
- Flaskの構築にあたって、MVTモデルやPRGパターンを採用し、開発保守をしやすい環境にすること
- Flaskの他のWebフレームワークにはない特徴的なリソースを最大限活用すること
- テンプレート(html)の装飾には "tailwind CSS" を採用し、モダンで魅力的な使いやすいデザインにすること
- 必要に応じてJavaScriptコードを作成してフロントエンドの側面でも開発保守しやすいようにすること

実行済みのシェルコマンド
```shell
pyenv local 3.12.3 # pyenvを使用してpythonのバージョンを指定
python -m venv .venv # 仮想環境の作成
source ./.venv/bin/activate # 仮想環境のアクティベート
pip install flask==2.3.3 # flaskをバージョン指定してインストール
```
